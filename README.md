#  Social Media Sentiment Analysis Dashboard

##  Project Overview
This project is an industry-level dashboard that analyzes social media sentiment using NLP techniques. It provides real-time insights through interactive visualizations.

---

##  Features
-  Interactive Dashboard (Streamlit)
-  FastAPI Backend (API Integration)
-  Word Cloud Visualization
-  Live Sentiment Prediction
-  Trend Analysis
- Spike Detection Alerts
-  Downloadable Reports

---

##  Tech Stack
- Python
- Streamlit
- FastAPI
- Pandas
- Plotly
- NLP (Text Processing)

---

##  Project Structure

app/ → Dashboard
api/ → Backend API
data/ → Dataset

---

## How to Run

### 1. Clone repo

### 2. Install dependencies

### 3. Run API

### 4. Run Dashboard
